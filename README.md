# Taxi App (Online Taxi)

A robust web application for booking and managing taxi rides, built using **Laravel**, **PHP**, and database systems like **MySQL/PostgreSQL**. This application provides a seamless workflow for passengers to book rides and for admins or drivers to manage operations via a secure dashboard.

## Key Features
*   **Passenger Booking:** Allows clients to easily request a taxi with specific car requirements.
*   **Driver Management:** Portal for handling driver registration and availability.
*   **Admin Dashboard:** Secure interface protected by authentication middleware to assign drivers to pending bookings.
*   **Localization Support:** Multi-language system supporting English (`en`), Turkmen (`tk`), and Russian (`ru`).

---

## Tech Stack
*   **Backend Framework:** Laravel (PHP)
*   **Frontend:** HTML5, CSS3, Bootstrap, Ajax
*   **Database:** MySQL / PostgreSQL
